Translator_XH – a CMSimple_XH Plugin

Translator_XH is an advanced tool for translating the CMSimple_XH core, pluginloader and plugins. It is mainly intended for translators, who want to publish their translations. If you only want to translate a few language strings, CMSimple_XH's built-in language forms should be sufficient.

Requirements

Translator_XH is a plugin for UTF-8 encoded CMSimple_XH ≥ 1.5. It requires PHP ≥ 5.1.0.

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory translator/ to your server into CMSimple_XH's plugins directory.
  4. Set write permissions for the subdirectories config/, css/ und languages/. The languages/ folders of the core, pluginloader and the plugins which shall be translated, need write permissions too.
  5. Switch to "Translator" in the back-end to check if all requirements are fulfilled.

Settings

The plugin's configuration is done as with many other CMSimple_XH plugins in the website's back-end. Select "Plugins" → "Translator".

You can change the default settings of Translator_XH under "Config". Hints for the options will be displayed when hovering over the help icons with your mouse.

Localization is done under "Language". You can translate the character strings to your own language, or customize them according to your needs. You might prefer to do this with the advanced facilities of Translator_XH – so have a look at Usage.

You can customize the look of Translator_XH under "Stylesheet".

back to top

Usage

In CMSimple_XH's back-end choose "Plugins" → "Translator" → "Translations" to see a list of all translatable modules. Just click one of the displayed modules to translate it – the translation form should be pretty much self explaining.

If you want to store the translated modules as properly arranged zip archive, just select the modules you want to include, enter the file name and press the save button.
